## Run Locally

Clone the project

```bash
  git clone https://github.com/shanukajain/hilarious-flame-4763.git
```

Go to the project directory

```bash
  cd hilarious-flame-4763
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
